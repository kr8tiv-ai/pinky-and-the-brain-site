**1. Create `src/components/dashboard/CommandHeader.tsx` as a Client Component:**

Add `'use client'` directive at top. Import `usePrice` from `@/hooks/usePrice`.

Define three local helper functions (NOT exported):
- `formatUsd(n: number): string` — if n < 0.01 return `$${n.toFixed(6)}`, else return `$${n.toLocaleString(undefined, { minimumFractionDigits: 2, maximumFractionDigits: 2 })}`
- `formatSol(n: number): string` — return the number to 8 decimal places followed by a space and the SOL symbol (unicode ◎)
- `formatChange(n: number): string` — prefix with + if >= 0, format to 2 decimal places, append %

Component implementation:
- Call `const { data, isLoading, isError } = usePrice()`
- Derive connection status from hook state: isError = "OFFLINE" (pink #ff9e9e), isLoading = "CONNECTING..." (muted #cccccc), data present = "LIVE" (accent #d4f000)
- Derive 24h change color: positive = `text-[#d4f000]`, negative = `text-[#ff9e9e]`, loading/zero = `text-[#cccccc]`

Layout structure — `<header>` element with two rows:
1. **Top bar** (`border-b border-[#333]/50`): Left side = "WAR ROOM — $BRAIN INTELLIGENCE DASHBOARD" in `font-mono text-xs uppercase tracking-[0.3em] text-[#cccccc] font-bold`. Right side = pulsing dot indicator + status label. The dot should be a 2x2 (w-2 h-2) rounded-full span with `bg-[#d4f000] animate-pulse shadow-[0_0_6px_#d4f000]` when connected, `bg-[#ff9e9e]` when error (no pulse). Use `px-6 lg:px-12 py-4` for padding.

2. **Data grid** (`grid grid-cols-2 md:grid-cols-5 divide-x divide-[#333]/50`): Five cells each with:
   - Label: `font-mono text-[10px] uppercase tracking-widest text-[#cccccc] mb-1 font-bold`
   - Value: `font-mono text-lg md:text-2xl font-black text-white tabular-nums`
   - Padding: `px-6 py-5`
   - Each cell:
     a. **PRICE (USD)**: `formatUsd(data?.priceUsd ?? 0)` or redacted `██████` blocks in `text-[#d4f000]` when loading
     b. **PRICE (SOL)**: `formatSol(data?.priceSol ?? 0)` or redacted blocks when loading
     c. **24H CHANGE**: `formatChange(data?.priceChange24h ?? 0)` with dynamic color, or redacted blocks in `text-[#cccccc]` when loading
     d. **MARKET CAP**: ALWAYS show `CLASSIFIED` badge since `data?.marketCap` is always undefined (Birdeye limitation). Use explicit null check: `data?.marketCap != null ? formatUsd(data.marketCap) : <span>`. The CLASSIFIED badge: `text-xs text-[#666] border border-[#333] px-2 py-0.5 tracking-widest font-mono`. Show redacted blocks when loading.
     e. **VOLUME 24H**: Same CLASSIFIED treatment as market cap. `data?.volume24h != null ? formatUsd(data.volume24h) : <span>`. Show redacted blocks when loading.

Header wrapper: `className="relative w-full border-b border-[#333] bg-[#0d0d0d]"` — full viewport width, no max-width constraint (command bar aesthetic).

IMPORTANT anti-patterns to avoid:
- Do NOT add `tabular-nums` without `font-mono` — they work together
- Do NOT show "$0" for marketCap/volume24h — always show CLASSIFIED
- Do NOT import HELIUS_RPC_URL or any server-only constants — only use the hook
- Do NOT add a noise-overlay div — the global one in layout.tsx already covers this page

Add a simple GSAP entrance animation: use `useRef` for the header container, `useEffect` with `gsap.context()` for cleanup. Animate `gsap.fromTo(headerRef.current, { y: -20, opacity: 0 }, { y: 0, opacity: 1, duration: 0.8, ease: 'power3.out' })`. Import gsap from 'gsap'. Return `() => ctx.revert()` from useEffect.
